# EduGuide

EduGuide is an AI-powered career guidance platform built using React and Vite. Designed to help students make academic and career decisions by providing stream recommendations, career roadmaps, entrance exam information, and college cutoff analysis.

## Features

* AI-powered career guidance assistant.
* Interactive stream selection quiz.
* Career roadmap and skill progression tracker.
* Entrance exams information portal.
* College cutoff trends and admission insights.
* Student dashboard for tracking progress.
* Login and Sign Up interface.
* Smooth animations and glassmorphism-inspired design.

## Tech Stack

* React
* Vite
* Tailwind CSS
* Framer Motion
* Lucide React
* JavaScript (ES6+)
