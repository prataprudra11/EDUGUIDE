# Product Requirements Document (PRD): EduGuide.ai

## 1. Product Vision & Objective
**EduGuide.ai** is an AI-powered career guidance platform designed to assist high school students and college aspirants in making informed academic and career decisions. The platform provides personalized stream recommendations, interactive career roadmaps, entrance exam tracking, and college cutoff analytics in a modern, engaging interface.

## 2. Target Audience
* **High School Students (Class 10-12):** Seeking clarity on stream selection and career paths.
* **College Aspirants:** Preparing for competitive entrance exams (e.g., JEE, NEET, CLAT) and researching college cutoffs.
* **Parents & Counselors:** Assisting students with data-driven academic planning.

## 3. Core Features & Capabilities

### 3.1 Authentication & User Profiles
* **Sign Up / Log In:** Secure user authentication.
* **Student Profile:** Captures basic details (Class, current stream, target roles).

### 3.2 Main Dashboard
* **Overview:** High-level summary of the student's progress.
* **Widgets:** Assessment status (0-100%), saved exams tracker, bookmarked colleges, and exam deadlines countdown.
* **Skill Tracker:** Active skills development graph over time.
* **Recommendations:** Suggested career portfolios based on profile.

### 3.3 Stream Selection Guidance (Psychometric Evaluation)
* **Interactive Quiz:** A multi-step psychometric questionnaire (e.g., activities, interests, aptitudes).
* **Analytics:** Generates a stream affinity radar chart (Analytical, Creative, Aptitude, etc.) matching the user to streams like Science Core, Commerce, etc.

### 3.4 Interactive Career Roadmap Builder
* **Role Selection:** Users can choose specific job roles (e.g., AI & Machine Learning Engineer).
* **Milestone Path Flowchart:** Step-by-step node timeline from Class 11/12 subjects to College Degrees, Skills, Internships, and Job Placement.
* **Active Stage Directives:** Detailed target skillsets and acquisition certifications for each milestone.
* **Salary Analytics:** Historical and projected salary growth trajectory graph.

### 3.5 Entrance Exams Hub
* **Search & Filter:** Find exams based on specialization (Engineering, Medical, Law) and difficulty.
* **Exam Cards:** Detailed views for exams (e.g., JEE Advanced) showing days left, eligibility, admission rules, and syllabus expansions.
* **Calendar Tracker:** Visual calendar highlighting exam dates and important milestones.

### 3.6 Colleges & Cutoff Analytics
* **Cutoff Search:** Filter by target branch, reservation category, and campus location.
* **Predictive Seat Calculator:** Enter mock rank/score to estimate admission probability.
* **Trend Graphs:** 5-year closing cutoff rank variations line chart.
* **Campus Catalogs:** Side-by-side comparison of colleges with NIRF rankings and average placement packages.

## 4. Non-Functional Requirements
* **Performance:** Fast load times (< 2s) and smooth micro-animations.
* **Design:** Premium, glassmorphism-inspired UI with Light/Dark mode support.
* **Responsiveness:** Seamless experience across desktop and tablet (mobile app version is planned for future).

## 5. Future Enhancements (Post-MVP)
* Real-time AI chatbot counselor.
* Native Mobile Application.
* Direct integration with college application portals.
